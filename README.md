# Road Traffic Analysis System

A real-time computer vision application that detects, tracks, and analyzes road traffic using a web-based user interface and a deep learning backend.

## 🚀 Features
* **Real-time Object Detection:** Detects and segments vehicles (cars, trucks, bikes) using the YOLOv4 framework.
* **Vehicle Tracking:** Implements object tracking to follow vehicles across video frames consistently.
* **Web-Based Interface:** A clean, user-friendly frontend built to handle video file uploads and visualize processing results.
* **Robust Backend:** A Flask-powered Python backend to manage video frames, coordinate files, and handle model inference.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** Python, Flask
* **Machine Learning & Computer Vision:** OpenCV, YOLOv4, Deep SORT
